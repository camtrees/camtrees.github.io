# {{ page.title }}

### An initial thought of what we will document in this website:

* Maintain a master "ToDo Task List" of short and long term needs/improvements so everyone involved in the CAMTREES Database project can provide direction and track updates.
* Show how volunteers use the [EpiCollect Phone App](https://five.epicollect.net) to collecct Chestnut Tree data while in the field.
* Discuss why we host the CAMTREES Database at [Neon.com](https://neon.com). And how admins, and volunteers can access the database.
* Document the Python programs to directly transfer EpiCollect records into the CAMTREES Database tables as well as to gather elevation data once a tree's GPS coordinates are available.
* Dream about how we might use [SoftR - A No-Code Web App Design System](https://www.softr.io) to create a web app which showcases CAMTREES Database tables and maps in real time. But realize that it may be too expensive to implement.
* How one could use the [R Statistical Program](https://www.r-project.org) to analyze the tree data.

We are in the very early stages of designing this website. We are currently creating a skeleton framework to house the various pages required.

{: .note}
We expect to document this database in fine detail. Both from a volunteer's perspective as well as a database admin's perspective.
