# {{ page.title }}

We are in the VERY EARLY days of developing this website. Both the design and the content are in flux. But, meanwhile,

### Some of the things we expect to include and document are:

* A master "ToDo Task List" of short and long term needs/improvements. This will allow everyone involved in the CAMTREES Database project the knowledge required so they can provide direction and track updates.
* Show how volunteers use the [EpiCollect Phone App](https://five.epicollect.net) to collecct Chestnut Tree data while in the field.
* Discuss why we host the CAMTREES Database at [Neon.com](https://neon.com). And how admins, and volunteers can access the database.
* Document all the Python programs used to manage our data needs.
* Document how one could use the [R Statistical Program](https://www.r-project.org) to analyze the tree data.
* Dream about how we might use [SoftR - A No-Code Web App Design System](https://www.softr.io) to create a web app which showcases CAMTREES Database tables and maps in real time. However, the SoftR system may not be affordable by the CAM project. That's why this bullet point might be a 'dream'.

We are currently creating a skeleton framework to house the various pages required.

{: .note}
Our goal is to document the CAMTREES Database for CAM Volunteers, CAM Staff, and CAMTREES Database Admins.
